# Abdulwahab Team Executive Dashboard

A premium executive dashboard built with Next.js and connected directly to Google Sheets.

## Live Data Source
- Spreadsheet ID: `1EwSWZ9XsyRY_HBNq9elmEsN35wbvGZZ12Rn_7FJGD8o`
- Sheet Name: `Adulwahab M Alshammari`

## Features
- Branch navigation
- Branch KPIs: Target, Achieved, Achievement %, Prep, Prep %, Post, Post %
- Employee ranking with progress bars
- Capture Mode
- Download PNG
- Download PDF
- Scalable metrics for future columns like 5G and Fiber

## Run locally
```bash
npm install
npm run dev
```

Then open:
```bash
http://localhost:3000/dashboard
```

## Important
Google Sheet access must be set to:
- Anyone with the link
- Viewer

## Deploy
This project is ready for deployment on Vercel.
